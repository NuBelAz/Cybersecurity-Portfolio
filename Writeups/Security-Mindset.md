# TryHackMe: Security Mindset - Attackers vs Defenders

**Completed:** April 2026  
**Status:** Complete

## What I Learned

This module taught me how to think like both an attacker and a defender:

**Attacker Mindset:**
- How attackers identify vulnerabilities and plan intrusions
- Common attack vectors and techniques
- Reconnaissance and enumeration methods
- Exploitation and post-exploitation tactics

**Defender Mindset:**
- How to anticipate and prevent attacks
- Defense-in-depth strategies
- Monitoring and detection techniques
- Incident response procedures

**Real-World Security:**
- How actual systems are attacked in practice
- Common security controls and how they work
- How defenders protect data and systems
- The cat-and-mouse game between attackers and defenders

## Key Takeaway

This module changed how I look at systems, now I see both opportunities (how attackers think) and protections (how defenders respond). Understanding the attacker mindset makes me a better defender. As a future Incident Responder, I need to think like an attacker to predict their moves, understand their techniques, and build effective defenses. I learned security isn't just about tools it's about mindset.
