# TryHackMe: Operating Systems Fundamentals

**Completed:** April 2026  
**Status:** Complete

## What I Learned

This module covered how users interact with operating systems through different interfaces:
- **Windows fundamentals**: File system structure, user permissions, graphical interface navigation, basic administration
- **Linux fundamentals**: Command line interface (CLI), basic commands (ls, cd, pwd, cat, grep), file permissions, directory structure
- **GUI vs CLI**: Understanding when to use graphical interfaces vs command line for system administration and security tasks

## Key Takeaway

As a future Security Analyst, I realized I need to be comfortable in both Windows and Linux environments since most enterprise networks use both. Learning the command line is especially critical for incident response. GUI tools can be slow or unavailable during investigations, but CLI access is always there. Understanding file permissions and system structure helps me identify where malware might hide or how attackers escalate privileges.

## Skills Gained
- Windows system navigation and administration basics
- Linux command line proficiency
- File system understanding (Windows vs Linux)

- Both GUI and CLI comfort

## Commands Learned (Linux)
- `ls` - list directory contents
- `cd` - change directory
- `pwd` - print working directory
- `cat` - view file contents
- `grep` - search text
- Basic file permissions understanding

## Connection to Security
- Understanding where log files are stored in both OS
- Knowing how to navigate systems during incident investigations
- Recognizing normal vs suspicious file locations
- Foundation for analyzing compromised systems
